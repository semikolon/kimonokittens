**Living with AI-aligned people could unlock:**
1. **Daily knowledge exchange** - breakfast conversations about prompt engineering

3. **Co-working synergies** - pair programming, code review, architectural discussions
4. **Startup incubation** - natural team formation from proven trust/collaboration
5. **Upward spirals** - mutual inspiration, accountability, momentum

## Target Tenant Profile: AI Kollektiv Edition

### Must-Haves

1. **Technical competence** - Can actually code (not just "interested in AI")
2. **AI tool fluency** - Uses Claude/GPT/Cursor daily, not experimentally
3. **Builder mentality** - Ships things, not just talks about them
4. **Collaborative spirit** - Excited to share knowledge, not hoard it
5. **Economic stability** - Can pay rent while building (savings, freelance, remote job)

### Ideal Signals

- Active GitHub with recent commits
- Personal projects using AI tools
- Writing/content about AI development
- Previous startup/small-team experience
- Generalist background (can work across stack)
- Domain expertise that complements (not duplicates) others

### Red Flags

- "I want to learn about AI" (consumer, not producer)
- No shipping history
- Pure theoretical interest without practical application
- Competitive/secretive about work
- Looking for cheap housing (not community investment)
